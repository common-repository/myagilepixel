Avoid legal issues with Google Analytics, Facebook Pixel, and TikTok Pixel. Boost marketing with custom user properties in Google Analytics 4.

== Description ==

This plugin helps you to prevent the illegal use of trackers on your website.

With different level of proxification, you can send anonymous data to USA and get compliant with GDPR.

You can continue to use Google Analytics, Facebook Pixel and TikTok Pixel and track your advertising campaign without worries about legal compliance.

With the added functionality of "User Property Tracking", you can enhance your web analysis and marketing campaigns by sending custom user properties to Google Analytics 4.

This software is compatible with Google Tag Manager and Google Ads Conversion Tracking.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin will:
1. Send your visitor tracking data with anonymous IP and without personal data
2. Helps you to stay GDPR compliant
3. Allows you to run Google Analytics, Facebook Pixel and TikTok Pixel without worries about legal compliance

== Installation ==

Installation is easy and fast. Just go to your WordPress dashboard, then Plugins > Add New.
From the search filter, just type My Agile Pixel. Press the install button and then activate the plugin.
